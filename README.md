# 🎵 Spoofity – Spotify Inspired Frontend UI

Spoofity is a frontend-only React.js project inspired by Spotify. It is a modern music-themed landing page UI featuring a hero section, playlist UI, recommendations, navbar with dropdown, and a floating mini player.

---

## 🚀 Features

- 🎨 Spotify-inspired UI design
- 🌈 Gradient background (Wine Red + Black theme)
- 🟢 Spotify green highlights (#1DB954)
- 📌 Navbar with dropdown menu:
  - Profile
  - Home
  - Playlist
  - Settings
- 🎧 Hero section with glowing text: "Feel The Music"
- 💡 Personalized recommendations section
- 🎵 Simple playlist UI
- 🎼 Floating mini music player
- 🌙 Dark modern aesthetic

---

## 🧱 Tech Stack

- React.js (Vite)
- JavaScript (JSX)
- Inline CSS / Styled UI Components
- Lucide React Icons (optional)
